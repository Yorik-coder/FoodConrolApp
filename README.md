FoodControl — это веб-приложение, которое помогает пользователям отслеживать свой рацион, контролировать калорийность и баланс нутриентов, а также достигать целей по здоровому питанию.

## ✨ Возможности

- 📝 Дневник питания с возможностью добавления приемов пищи
- 🔍 Поиск продуктов по базе данных (или ручное добавление)
- 📊 Подсчет калорий, белков, жиров и углеводов (КБЖУ)
- 📈 Графики прогресса за день/неделю/месяц
- 🎯 Установка индивидуальных целей (потеря веса, поддержание, набор массы)
- 👤 Профиль пользователя с настройками
